#

거버넌스 토큰 보유자는 다음과 같이 즉각적이고 무효화할 수 없는 통제권을 가집니다.

* 커뮤니티 트레저리 할당
* 프로토콜 신규 토큰 상장
* 프로토콜의 리스크 파라미터
* 유동성 스테이킹 풀의 마켓 메이커에 대한 자본 할당

* 유동성 스테이킹 풀에 신규 마켓 메이커 추가

* 손실이 발생할 경우 안전 스테이킹 풀 배당금 지급 결정
* 출시 시 존재하는 모든 보상 및 풀의 변경
* 거버넌스 자체 계약

dYdX 거버넌스는 다음 계약의 파라미터에 대한 통제 권한을 갖습니다.

* [타임락](https://github.com/dydxfoundation/governance-docs/tree/28153eacbdaafb32078630fafa7ad64f111ac9ab/voting-and-governance-process/parameters.md#timelock-parameters)
* 우선 타임락
* 거버너
* $ethDYDX 토큰
* 트레저리
* 머클 분배자
* 유동성 스테이킹
* 안전 모듈
* 스타크 프록시
* 스타크 퍼페츄얼

## 스타크 프록시

| 매개 변수 | 설명 | 값 |
| ------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------- |
| 소유자 역할 | 자금 + STARK 키를 수령한 수령자를 추가 및 제거할 수 있고 유동성 스테이킹 및 Stark 무기한 계약에 ERC20 수당을 설정할 수 있으며 강제 조치 실행할 수 있음. 소유자의 관리자이자 위임 관리자 역할 수행 | 마켓 메이커 |
| 위임 관리자 역할 | 차용자, 거래소 운영자 및 출금 운영자 역할의 관리자 | 마켓 메이커 |
| 차용자 역할 | 유동성 스테이킹 계약에 차입 기능을 호출할 수 있음 | 마켓 메이커 |
| 거래소 운영자 역할 | Stark 무기한 계약에 교환 기능을 호출할 수 있음 | 마켓 메이커 |
| 출금 운영자 역할 | 허용 수령자에게 차입 잔액을 초과하는 자금을 인출할 수 있음 | 마켓 메이커 |
| 보호자 역할 | 종료 조치를 수행하고 차용자에게 연체된 부채가 있는 경우 강제 조치를 수행하고 차용 자금으로 미결 조치를 제한하며, 인출 운영자 역할에 의해 외부에서 인출할 토큰 금액을 승인할 수 있음 | 단기 타임락 |
| 거부권 보호자 역할 | 대기 기간에 소유자가 시작한 강제 거래 요청을 거절할 수 있음 | 머클 일시정지자 타임락 |
